# Встановлення пакетів Supabase

## Проблема
npm не знайдено в PATH системи.

## Рішення

### Варіант 1: Встановити Node.js (рекомендовано)
1. Завантажте Node.js з офіційного сайту: https://nodejs.org/
2. Встановіть Node.js (npm встановлюється автоматично)
3. Перезапустіть термінал/IDE
4. Виконайте команду:
   ```bash
   npm install @supabase/supabase-js @supabase/ssr
   ```

### Варіант 2: Використати інший термінал
Якщо Node.js вже встановлений, але не доступний в поточному терміналі:
1. Відкрийте новий термінал (Command Prompt або PowerShell)
2. Перейдіть до папки проєкту:
   ```bash
   cd "G:\My Drive\!Hero rooms\v3 (1)"
   ```
3. Виконайте команду:
   ```bash
   npm install @supabase/supabase-js @supabase/ssr
   ```

### Варіант 3: Використати VS Code Terminal
1. Відкрийте проєкт у VS Code
2. Відкрийте вбудований термінал (Ctrl + `)
3. Виконайте команду:
   ```bash
   npm install @supabase/supabase-js @supabase/ssr
   ```

### Варіант 4: Перевірити PATH
Якщо Node.js встановлений, але npm не працює:
1. Знайдіть шлях до Node.js (зазвичай: `C:\Program Files\nodejs\`)
2. Додайте його до системного PATH:
   - Відкрийте "Системні змінні середовища"
   - Додайте шлях до Node.js до змінної PATH
   - Перезапустіть термінал

## Після встановлення

Після успішного встановлення пакетів:
1. Запустіть додаток: `npm run dev`
2. Тестова сторінка Supabase відкриється автоматично
3. Перевірте статус підключення

## Перевірка встановлення

Після встановлення перевірте, що пакети додані:
```bash
npm list @supabase/supabase-js @supabase/ssr
```

Або перевірте файл `package.json` - там повинні бути додані залежності.
