# 🗂️ Структура проекту - Візуальний гайд

## 📁 Як організований ваш проект

```
v3 (1)/                          ← Головна папка проекту
│
├── 📄 App.tsx                    ← 🎯 ГОЛОВНИЙ ФАЙЛ
│                                  Тут визначається:
│                                  - Які сторінки показувати
│                                  - Навігація між сторінками
│                                  - Загальна структура сайту
│
├── 📄 index.tsx                  ← Точка входу (запускає App.tsx)
├── 📄 index.html                 ← HTML шаблон
├── 📄 index.css                  ← Глобальні стилі
│
├── 📁 components/                ← 🎨 ВСІ КОМПОНЕНТИ (частини сайту)
│   │
│   ├── 📄 Navbar.tsx             ← Навігація зверху
│   │                              (Dashboard, Marketplace, Account)
│   │
│   ├── 📄 FilterBar.tsx          ← Фільтри для пошуку нерухомості
│   │
│   ├── 📄 PropertyCard.tsx       ← 🏠 Карточка нерухомості в списку
│   │                              (маленька карточка з фото та ціною)
│   │
│   ├── 📄 PropertyDetails.tsx    ← 📋 Детальна інформація про нерухомість
│   │                              (велика панель справа з усіма деталями)
│   │
│   ├── 📄 BookingForm.tsx        ← 📝 Форма для бронювання
│   │
│   ├── 📄 Marketplace.tsx        ← 🏪 Маркетплейс (ринок нерухомості)
│   │
│   ├── 📄 AccountDashboard.tsx   ← 👤 Панель акаунту
│   │
│   ├── 📄 PartnerModal.tsx       ← 🤝 Модальне вікно "Стати партнером"
│   │
│   └── ... (інші компоненти)
│
├── 📄 constants.ts               ← 📊 ДАНІ ПРОЕКТУ
│                                  Тут зберігається:
│                                  - Список нерухомості (MOCK_PROPERTIES)
│                                  - Дані компаній
│                                  - Маркетплейс оголошення
│
├── 📄 types.ts                   ← 📝 ТИПИ ДАНИХ
│                                  Описує структуру даних:
│                                  - Що таке Property (нерухомість)
│                                  - Що таке Booking (бронювання)
│                                  - Інші типи
│
├── 📄 package.json               ← ⚙️ НАЛАШТУВАННЯ ПРОЕКТУ
│                                  - Список залежностей
│                                  - Команди (npm run dev, npm run build)
│
└── 📄 vite.config.ts             ← 🔧 КОНФІГУРАЦІЯ ЗБІРКИ
```

---

## 🎯 Основні файли для редагування

### 1. `constants.ts` - Зміна даних
**Що тут:** Список всієї нерухомості, компаній, оголошень

**Що можна зробити:**
- ✅ Додати нову нерухомість
- ✅ Змінити ціни
- ✅ Змінити описи
- ✅ Додати нові фото
- ✅ Змінити характеристики

**Приклад:**
```typescript
{
  id: '1',
  title: 'Моя квартира',        ← Змініть тут
  price: 750,                    ← Змініть тут
  address: 'Шевченка 15А',       ← Змініть тут
  // ... інші поля
}
```

---

### 2. `components/PropertyCard.tsx` - Дизайн карточки
**Що це:** Як виглядає маленька карточка нерухомості в списку

**Що можна зробити:**
- ✅ Змінити розмір карточки
- ✅ Змінити кольори
- ✅ Додати нові поля (наприклад, рейтинг)
- ✅ Змінити розташування елементів

---

### 3. `components/PropertyDetails.tsx` - Детальна інформація
**Що це:** Велика панель з усіма деталями про нерухомість

**Що можна зробити:**
- ✅ Додати нові секції (наприклад, "Відгуки")
- ✅ Змінити макет
- ✅ Додати нові кнопки
- ✅ Змінити відображення фото

---

### 4. `App.tsx` - Головна логіка
**Що це:** Мозок сайту - визначає що показувати

**Що можна зробити:**
- ✅ Додати нові сторінки
- ✅ Змінити навігацію
- ✅ Додати нові розділи
- ✅ Змінити загальну структуру

---

## 🔄 Як працює сайт (простими словами)

```
1. Користувач відкриває сайт
   ↓
2. Завантажується index.html
   ↓
3. Запускається index.tsx
   ↓
4. Завантажується App.tsx
   ↓
5. App.tsx визначає: "Показати Dashboard"
   ↓
6. Завантажуються компоненти:
   - Navbar (навігація)
   - FilterBar (фільтри)
   - PropertyCard (карточки)
   - PropertyDetails (деталі)
   ↓
7. Дані беруться з constants.ts
   ↓
8. Все відображається на екрані!
```

---

## 📝 Приклад: Як додати нову нерухомість

### Крок 1: Відкрийте `constants.ts`

### Крок 2: Знайдіть `MOCK_PROPERTIES`

### Крок 3: Додайте новий об'єкт:
```typescript
{
  id: '9',  // Унікальний номер
  title: 'Нова квартира',
  address: 'Нова адреса 123',
  city: 'Київ',
  price: 1000,
  rooms: 2,
  area: 50,
  // ... інші поля (скопіюйте з існуючого)
}
```

### Крок 4: Збережіть файл (`Ctrl + S`)

### Крок 5: Подивіться в браузер - нова нерухомість з'явиться!
