# ⚠️ Проблема: Git не знайдено

Git не знайдено в системі або не додано до PATH.

## Швидке рішення

### Варіант 1: Перезапустити Cursor (найпростіше)
1. Закрийте Cursor повністю
2. Відкрийте Cursor знову
3. Відкрийте Terminal (`Ctrl + ~`)
4. Спробуйте `git --version`

Якщо після перезапуску Git працює - продовжіть з планом.

### Варіант 2: Використати GitHub Desktop
Якщо у вас встановлений GitHub Desktop:
1. Відкрийте GitHub Desktop
2. File → Add Local Repository
3. Виберіть папку: `G:\My Drive\!Hero rooms\v3 (1)`
4. Натисніть "Publish repository"
5. Введіть назву репозиторію
6. GitHub Desktop автоматично використає ваш токен

### Варіант 3: Встановити Git заново
1. Завантажте Git: https://git-scm.com/download/win
2. Встановіть з налаштуваннями за замовчуванням
3. Перезапустіть Cursor
4. Спробуйте `git --version`

## Після вирішення проблеми

Повідомте мене, коли Git працює, і я продовжу з планом підключення до GitHub.
